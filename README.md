# lenta_tech

## 1️⃣ Подготовка данных

* Разбивка видео/фотодатасета по `timestamp`.
* Усреднение нескольких кадров вокруг нужного timestamp или выбор лучшего кадра.

## 2️⃣ Детекция ценников (vbox)

* Дообучение YOLOv8 (nano/s/m) для локализации ценников.
* Получение bbox каждого ценника на кадре.

## 3️⃣ Обработка дубликатов

* Один и тот же ценник может появляться на разных кадрах.
* Группировка по уникальному `id` (tracking/по bbox+IoU).
* Для инференса выбирается наиболее часто встречающийся результат по полям.

## 4️⃣ Определение типа ценника

* Для каждого bbox определить тип ценника (7 типов).

* **Набор правил (по внешнему виду)**:

  1. Не соответствует типам 2–6 → тип 1.
  2. Не 3–6 и есть кружок с буквой → тип 2.
  3. Есть прямоугольник → тип 3.
  4. Нет штрихкода → тип 4.
  5. Штрихкод есть, но не в правом нижнем углу → тип 5.
  6. Прямоугольник → тип 6.

* Возможный вариант: нейросеть‑классификатор для автоматической классификации типов.

## 5️⃣ Локализация полей внутри ценника

* Для каждого типа выделяются ключевые компоненты: цена, валюта, акции и т. д.
* Подходы:

  * Правила по геометрии и положению элементов.
  * Обученная модель детекции полей.

## 6️⃣ OCR

* EasyOCR для распознавания текста каждого компонента.

## 7️⃣ Агрегация результатов

* Сгруппировать распознанные поля по `id` ценника.
* Выбрать наиболее частые или качественные значения для итогового результата.


## 🔧 Стек технологий

| Компонент       | Технологии                                            |
| --------------- | ----------------------------------------------------- |
| Backend         | Python 3.8+, FastAPI                                  |
| Детекция        | PyTorch, Ultralytics YOLOv8 (detect + optional track) |
| OCR             | EasyOCR (или PaddleOCR)                               |
| Frontend        | HTML, JavaScript, CSS                                 |
| Data processing | OpenCV, NumPy                                         |
| Deployment      | Docker (опционально)                                  |
